
# TUS.Float

TUS.Float is jquery plugin to except only float value.
[coffee for me?](https://www.buymeacoffee.com/uekichinos)

## Plugin Parameters

1.  **decimal** - decimal point, by default 2.

## Sample Code

Sample code as below.
```
$("#float").keyup(function(){
	$(this).tusfloat({
		decimal: 2
	});
});
```
